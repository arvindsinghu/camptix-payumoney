PayU Money Payment Gateway for CampTix plugin

== Description ==

PayU Money (previously PayU Paisa) Payment Gateway for the CampTix plugin.

Accpet payments in INR through PayUmoney using the CampTix plugin. 

== Installation ==

1. Upload `camptix-payupaisa-gateway` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Tickets -> Setup` in your WordPress admin area to set the currency to INR and activate the PayU Paisa gateway.
